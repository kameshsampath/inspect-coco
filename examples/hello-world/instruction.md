## Goal

Create a file `/workspace/hello.txt` containing exactly the text "Hello, World!".

## Requirements

- The file must be created at the exact path `/workspace/hello.txt`
- The content must be exactly "Hello, World!" with no trailing newline
- The file must be a plain text file (not binary)

## Constraints

- Do not create any other files
- Do not modify any existing files in the workspace
- Do not install any packages

## Output

Success criteria:
- File `/workspace/hello.txt` exists
- Content is exactly "Hello, World!" (verified by test script)
