## Goal

Create a Python module `/workspace/mathlib.py` with basic math functions.

## Requirements

- Must contain a function `add(a, b)` that returns the sum
- Must contain a function `subtract(a, b)` that returns the difference
- Must contain a function `multiply(a, b)` that returns the product
- All functions must handle integer and float inputs
- All functions must raise `TypeError` for non-numeric inputs

## Constraints

- Do not use any external libraries
- Do not create any files other than `/workspace/mathlib.py`
- Do not add a `__main__` block

## Output

Success criteria:
- `/workspace/mathlib.py` exists
- All pytest tests in `/workspace/tests/test_mathlib.py` pass
