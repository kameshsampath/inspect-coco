# inspect-coco

Inspect AI evaluations for Cortex Code (CoCo) agents and skills.

## Overview

`inspect-coco` enables CoCo skill developers to write, run, and score deterministic evaluations of their skills using [Inspect AI](https://inspect.aisi.org.uk/) as the orchestration framework.

## Installation

```bash
uv add git+https://github.com/kameshsampath/inspect-coco.git
```

## Quick Start

```bash
inspect eval examples/hello-world/task.py
```
